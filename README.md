# Breeze - Weather App

Breeze is a modern, responsive weather dashboard built with React and Vite. It features real-time weather updates, city searches, and a sleek glassmorphism UI.

## Features
- Real-time weather data
- Interactive Map view
- Theme switching (Light/Dark mode)
- Responsive design for all devices

## Technologies Used
- React
- Vite
- Framer Motion
- Lucide React
- React Router DOM
- CSS Variables for Theming
